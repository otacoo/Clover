4chanX → Clover Filter Converter

iThis tool converts 4chanX text-based filters into Clover's JSON filter format and then merges with your Clover settings backup, leaving both original files intact.

Converted: general, comment, subject, name, tripcode, unique ID, flag, filename.
Skipped: MD5, dimensions, filesize, email, pass, post ID, capcode.
